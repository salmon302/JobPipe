# JobPipe Extension Icons

This directory should contain the following icon files for the browser extension:

- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon48.png` - 48x48 pixels (extension management page)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Quick Icon Generation

You can create simple placeholder icons using any image editor or online tool.

For testing purposes, you can:
1. Create simple PNG files with the text "JP" (JobPipe)
2. Use a tool like https://favicon.io/ to generate icons
3. Use any 512x512 icon and resize it to the required dimensions

## Icon Design Suggestions

- Use a wave/water-themed icon (referencing "Pipe")
- Colors: Purple-to-blue gradient (#667eea to #764ba2)
- Simple, recognizable at small sizes
- Transparent background (PNG format)
